Student Grade Tracker

Overview

Student Grade Tracker is a Java-based application developed as part of the CodeAlpha Java Programming Internship. The project helps manage student records, store marks, calculate grades, and display academic performance.

Features

- Add student details
- Enter student marks
- Calculate grades automatically
- Display student results
- Validate user input
- Menu-driven console interface

Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Conditional Statements
- Loops and Arrays

How to Run

1. Download or clone the repository.
2. Open the project in any Java IDE (IntelliJ IDEA, Eclipse, NetBeans, or VS Code).
3. Compile the Java file.
4. Run the program.
5. Follow the on-screen instructions.

Sample Functions

- Add Student
- Enter Marks
- Calculate Grade
- Display Result
- Exit Program
